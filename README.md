### Open source

- **[leadguard](https://github.com/valterjuniorsilv/leadguard)**
  Pega lead do WhatsApp e joga no CRM sem duplicar. TypeScript, Clean Architecture e DDD. Tem lock distribuído em Postgres porque cliente manda quatro mensagem em duzentos milissegundos e o sistema tem que aguentar.

- **[claude-skills](https://github.com/valterjuniorsilv/claude-skills)**
  Treze skills que eu uso todo dia no Claude Code. Brutal honesty mode (já cansei de bajulação), pipeline de copy em PT-BR e umas pra retomar sessão de onde parei.

- **[claude-whatsapp-template](https://github.com/valterjuniorsilv/claude-whatsapp-template)**
  Bot de WhatsApp pronto pra clonar. Já vem com o que ninguém documenta: debounce de mensagem em sequência, lista de contatos protegidos e fallback quando o Claude trava.

- **[local-business-prospector](https://github.com/valterjuniorsilv/local-business-prospector)**
  Scraper de Google Maps com enriquecimento de email, Instagram e anúncio. Score do lead via Claude. Sai por uns dois reais e meio cada mil leads.

- **[agency-as-agents](https://github.com/valterjuniorsilv/agency-as-agents)**
  Oito agents do Claude Code montados igual organograma de agência pequena. Cinco de engineering, três de comercial.

- **[antigravity-lab](https://github.com/valterjuniorsilv/antigravity-lab)**
  Backend em Go que tô usando pra estudar Clean Architecture, DDD e CQRS de cabeça.
