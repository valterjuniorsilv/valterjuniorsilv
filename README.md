Trabalho com marketing há uns 5 anos. No último, cansei de esperar dev pra subir coisa simples e fui aprender a programar. Hoje toco quase tudo em Python, TypeScript e n8n, com Claude no meio quando faz sentido.

### Open source

- **[leadguard](https://github.com/valterjuniorsilv/leadguard)** — pega lead do WhatsApp e joga no CRM sem duplicar nem perder. TypeScript com Clean Architecture e DDD de verdade. Lock distribuído em Postgres porque cliente manda 4 mensagem em 200ms e o sistema precisa aguentar.
- **[claude-skills](https://github.com/valterjuniorsilv/claude-skills)** — 13 skills que eu uso todo dia no Claude Code. Tem brutal honesty mode (pra ele parar de me bajular), pipeline de copy PT-BR (pra texto não sair com cara de IA), e umas pra retomar sessão de onde parei.
- **[claude-whatsapp-template](https://github.com/valterjuniorsilv/claude-whatsapp-template)** — boilerplate de bot WhatsApp pronto. Já vem com as coisas chatas resolvidas: debounce de mensagem em sequência, contatos protegidos (pra ele não atender minha mãe), fallback quando a API do Claude cai.
- **[local-business-prospector](https://github.com/valterjuniorsilv/local-business-prospector)** — scraper de Google Maps + enriquecimento de email/IG/anúncio + score do lead via Claude. Sai por uns R$ 2,50 cada mil leads tratados.
- **[agency-as-agents](https://github.com/valterjuniorsilv/agency-as-agents)** — 8 agents do Claude Code montados igual organograma de agência pequena. Engineering e comercial.
- **[antigravity-lab](https://github.com/valterjuniorsilv/antigravity-lab)** — backend Go que tô usando pra estudar Clean Architecture + DDD + CQRS de cabeça.
